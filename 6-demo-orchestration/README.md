# Démonstration d'orchestration manuelle

## Objectifs

Installer l'application *spring-postgres* de l'introduction avec des docker runs à partir du README.md en 10 minutes.

## Etapes
### 1. Lire le README.md.
### 2. Construire l'image backend à partir du Dockerfile
### 3. Lancer la base Postgres avec les bons paramètres
### 4. Trouver les bons paramètres pour le backend (application.properties)
### 5. Lancer le backend avec les bons paramètres

### À observer
- Le processus de déploiement est long et fastidieux
- Je dépends du README.md et de ma connaissance de l'application
- Il me faudrait scripter les docker runs pour partager ma connaissance sur ce déploiement.
